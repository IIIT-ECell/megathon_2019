PROBLEM STATEMENTS '24

PROBLEM 1

Predicting Natural Disasters Using Satellite Data

Create a model that identifies patterns or risk factors indicating the onset of natural disasters like floods and landslides. Use satellite imagery and relevant data to predict these disasters, aiming for early detection and prediction to minimize impact.

PROBLEM 2

Yoga Instructor Mobile Application

Develop a mobile application that guides users through yoga poses using a body pose estimation model. The app should provide real-time feedback, comparing user movements to a "golden video" and generating alerts for incorrect postures. Hardware acceleration (NPU preferred, GPU fallback) is essential.

PROBLEM 3

MindPeers Problem Statement

Develop an NLP-based solution to automatically extract, classify, and analyze mental health concerns from user input. The solution should include polarity detection, keyword extraction (NER), concern classification, intensity scoring, and a timeline-based sentiment shift analysis.
